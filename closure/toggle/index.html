Toggle Function Demo

Problem Statement

Create a toggle function that accepts a list of arguments and toggles each of them when invoked in a cycle.

let hello = toggle("hello");
hello() // "hello";
hello() // "hello";

let onOff = toggle("on", "off");
onOff() // "on"
onOff() // "off"
onOff() // "on"

Demo 1: Single Argument

When only one argument is provided, it should always return the same value.

Demo 2: Two Arguments (On/Off Toggle)

Alternates between "on" and "off".

Demo 3: Multiple Arguments (Colors)

Cycles through red, green, and blue.

Demo 4: Numbers

Cycles through numbers 1, 2, 3, 4.

Demo 5: Mixed Types

Cycles through different data types: boolean, string, number, null.
